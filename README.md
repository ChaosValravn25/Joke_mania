# Joke_mania - App de Chistes por Categoría
- Joke_mania es una prueba de concepto desarrollada en Flutter como parte de la Evaluación Acumulativa Opcional del curso Programación de Dispositivos Móviles 2025/1. Permite obtener chistes aleatorios por categoría, seleccionarla como favorita, compartir el chiste y navegar entre pantallas informativas.
